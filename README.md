## Prerequisites

**Rust** 1.70.x - 1.75.x (required for Bevy 0.12.0)  
**Cargo** 1.70.x - 1.75.x (included with Rust)

## Installation

```bash
git clone https://github.com/louis-jean0/boidsy.git
cd boidsy
cargo build
cargo run
```
